# ARCHITEK PRO

Encyclopédie personnelle d'architecture d'intérieur.

App web minimaliste — HTML / CSS / JS pur.

## Fonctionnalités

- 8 domaines, 50+ fiches pré-remplies
- Matériaux classés par catégories (Bois, Pierres, Enduits, Métaux, Textiles, Céramique, Innovants)
- Suivi de progression (lu / maîtrisé / favori)
- Notes personnelles, liens, images par fiche
- Conseils pratiques avec espace personnel
- Recherche instantanée
- 100% localStorage — aucun serveur

## Lancer

Ouvrir `index.html` dans un navigateur.
